# Plush Petals - Luxury Artificial Flowers

**⚠️ PRIVATE WEBSITE - FOR PERSONAL/COMMERCIAL USE ONLY**
*This is a personal store website. Unauthorized reproduction or use is prohibited.*

A beautiful, responsive website showcasing luxury artificial flowers crafted with velvet and plush materials.

## 🌸 Features

- **Product Gallery**: 6 stunning assortments with high-quality images
- **Interactive Previews**: Detailed product pages with full-size images
- **Order System**: Multiple contact methods including WhatsApp integration
- **Responsive Design**: Optimized for all devices (mobile, tablet, desktop)
- **Modern UI**: Clean, elegant design with smooth animations
- **Contact Form**: Integrated contact system with EmailJS

4. Open http://localhost:8000 in your browser
